--- layout: post title: 'Quick tip: concatenate multiple CSV files' date: '2012-12-21T00:00:00+00:00' permalink: "/quick-tip-concatenate-multiple-csv-files/" --- If you're importing sales, oftentimes you need to import more than one CSV file at a time. Here's a shortcut for the Mac to help you combine multiple files into one new one. Create a folder on your desktop and put the csv files in it. Open a Terminal window and type the following: cat *.csv > my-new-file.csv Immediately you'll have a new file called my-new-file, which contains the content of all the other files in that folder. This works for data files other than CSV.
